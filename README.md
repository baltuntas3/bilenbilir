# Bilen Bilir

A real-time multiplayer trivia game where players compete by answering questions in hosted rooms.

**Live:** https://bilenbilir-web-fxaftzp5aa-ew.a.run.app/

## Tech Stack

- **Frontend:** React + Vite, Mantine UI
- **Backend:** Node.js, Socket.IO
- **Deployment:** Docker, Google Cloud Run

## Getting Started

```bash
# Backend
cd backend && npm install && npm run dev

# Frontend
cd frontend && npm install && npm run dev
```
